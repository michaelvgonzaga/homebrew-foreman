# homebrew-foreman

Homebrew tap for [Foreman](https://github.com/michaelvgonzaga/foreman) — a Claude Code framework for structured, verified AI-assisted development.

## Install

```bash
brew tap michaelvgonzaga/foreman
brew install foreman-ai
```

Then launch from any directory:

```bash
foreman-ai
```

On first run it creates a `foreman/` folder in your current directory and opens it in Claude Code.

## Update

```bash
brew upgrade foreman-ai
```

## Uninstall

```bash
brew uninstall foreman-ai
brew untap michaelvgonzaga/foreman
```

## Requirements

- [Claude Code](https://claude.ai/code)
- Git
